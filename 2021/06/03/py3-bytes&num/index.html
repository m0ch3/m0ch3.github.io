struct模块介绍

python中的struct模块主要用于处理C结构数据和Python数据类型互转，类似于json。
struct.pack(fmt,v1,v2,.....)：将v1,v2,…（python类型）按照fmt格式化字符串指定的格式转化成C语言底层存储结构，在python3中的表现为，返回一个字节串（b’…’)。
struct.unpack(fmt,bstring)：pack()的逆方法，将bstring字节串按照fmt指定的格式转换成python类型，返回值是一个tuple（即使是单一返回值也是一个tuple）。需要注意的是，bstring的len要和struct.calcsize(fmt)相等。
fmt格式串的参照规则如下：

Format C Type Python
x pad type no value
c char str (len=1)
b signed char int
B unsigned char int
? _Bool bool
h short int
H unsigned short int
i int int
I unsigned int int
l long int
L unsigned long int
q long long int
Q unsigned long long int
f float float
d double float
s char[] str
p char* str
P void* int

在fmt的首位，有一个可选标记决定大小端序：

sign order
@ native
= native
< little-endian
> big-endian
! network(= big-endian)
